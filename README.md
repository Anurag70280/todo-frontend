# Todo Frontend (React + TypeScript)

A simple and responsive Todo web app built using **React**, **TypeScript**, and **Axios** for API integration.  
This frontend communicates with the **Todo Backend** (Node.js + Express + MongoDB) for user authentication and task management.

---

## Features

-  User login & signup (JWT-based authentication)
-  Add, edit, delete, and mark tasks as complete/incomplete
-  Search & filter tasks by status
-  Responsive design that works across all devices
-  Axios integration for real-time backend updates

---

##  Tech Stack

- **React + TypeScript**
- **Axios** for API requests
- **React Icons** for UI elements
- **CSS Modules / Custom CSS**
- **Vite** as the build tool
